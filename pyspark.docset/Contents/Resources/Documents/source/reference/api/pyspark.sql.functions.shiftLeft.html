pyspark.sql.functions.shiftLeft¶

pyspark.sql.functions.shiftLeft(col, numBits)[source]¶

Shift the given value numBits left.

>>> spark.createDataFrame([(21,)], ['a']).select(shiftLeft('a', 1).alias('r')).collect()
[Row(r=42)]

New in version 1.5.
