pyspark.sql.DataFrame.printSchema¶

DataFrame.printSchema()[source]¶

Prints out the schema in the tree format.

>>> df.printSchema()
root
 |-- age: integer (nullable = true)
 |-- name: string (nullable = true)

New in version 1.3.
