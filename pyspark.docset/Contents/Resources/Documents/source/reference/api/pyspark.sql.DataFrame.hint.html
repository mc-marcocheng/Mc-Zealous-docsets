pyspark.sql.DataFrame.hint¶

DataFrame.hint(name, *parameters)[source]¶

Specifies some hint on the current DataFrame.

Parameters

  • name – A name of the hint.

  • parameters – Optional parameters.

Returns

DataFrame

>>> df.join(df2.hint("broadcast"), "name").show()
+----+---+------+
|name|age|height|
+----+---+------+
| Bob|  5|    85|
+----+---+------+

New in version 2.2.
